# QUICKBITE
<p> Developed a food ordering platform with 3 user roles (Customer, Restaurant Owner,
and Admin), supporting restaurant discovery and order management.</p> 
<p> Implemented search, filtering, and review functionalities across 10+ restaurants and 20+ menu items
to improve food discovery and user engagement.</p> 
<p> Built cart management, order placement, and order history tracking workflows, providing a seamless
food ordering experience.</p> 
<p> Used Redis for JWT token management and enabled Restaurant Owners to manage restaurants, menus,
and incoming orders through a dedicated dashboard.</p>
